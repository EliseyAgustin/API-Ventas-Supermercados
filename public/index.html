API de Ventas de Supermercados

Sistema de gestión de datos de ventas de supermercados

Endpoints Disponibles

Endpoints Básicos

  • GET /api/ventas - Obtener todos los registros
  • GET /api/ventas/:fecha - Obtener registro por fecha
  • POST /api/ventas - Crear nuevo registro
  • PUT /api/ventas/:fecha - Actualizar registro existente
  • DELETE /api/ventas/:fecha - Eliminar registro

Filtros y Funcionalidades Adicionales

  • GET /api/ventas/filtro/fecha?desde=YYYY-MM-DD&hasta=YYYY-MM-DD - Filtrar por rango de fechas
  • GET /api/ventas/filtro/canal?canal=online|salon - Filtrar por canal de venta
  • GET /api/ventas/filtro/medio-pago?medio=efectivo|tarjetas_debito|tarjetas_credito|otros_medios - Filtrar por medio de pago
  • GET /api/ventas/filtro/categoria?categoria=bebidas|almacen|... - Filtrar por categoría de producto
  • GET /api/estadisticas - Obtener estadísticas generales
